ZigBee-CAN Monitoring System

Hybrid wired-wireless real-time data acquisition and remote monitoring

📌 Overview

This project implements a hybrid communication system that integrates the robustness of the CAN Bus protocol with the wireless flexibility of ZigBee. It enables real-time monitoring of sensor data, bypassing the physical limitations of purely wired networks.

⚙️ Objectives

  • Design a reliable CAN Bus communication network
  • Integrate ZigBee for remote wireless data monitoring
  • Implement low-power real-time data acquisition
  • Minimize cost using lab-available X2C ZigBee modules

🧠 Methodology

Ultrasonic sensor data is processed via ESP32 and transmitted over a CAN network using MCP2515 controllers. A central node interfaces the CAN data with a ZigBee module (configured via Arduino Uno) for wireless transmission to a remote station.

🛠️ Technologies Used

  • ESP32 Microcontrollers (UART/SPI/I2C)
  • MCP2515 CAN Bus Modules
  • X2C ZigBee Modules & Shields
  • ESP-IDF Development Framework

🚀 Features

  • Hybrid wired-wireless architecture
  • Real-time sensor data acquisition
  • Scalable and low-power system design
  • Integrated local LCD display

📊 Applications

  • Automotive diagnostics
  • Industrial monitoring systems
  • Smart grid systems
  • IoT-based remote environmental monitoring
